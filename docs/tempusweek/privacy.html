Política de privacidad

Última actualización: julio de 2026

Esta política de privacidad describe cómo la aplicación TempusWeek, desarrollada por TONIMED TECH, recoge, usa y protege la información personal de sus usuarios.

1. Responsable

El responsable del tratamiento de los datos es TONIMED TECH, marca bajo la que se desarrolla esta aplicación de uso personal y educativo. Para cualquier consulta relacionada con privacidad, puedes escribir a toniketrona8@gmail.com.

2. Datos que se recogen

La aplicación recoge y almacena únicamente los datos que tú introduces:

  • Dirección de email — usada como identificador de cuenta.
  • Tareas y categorías — título, descripción, fecha, horas de inicio/fin, categoría y regla de recurrencia.

La aplicación no recoge datos de geolocalización, contactos, cámara ni ninguna información no relacionada con la planificación de tareas.

3. Dónde se almacenan los datos

Los datos se almacenan principalmente en el propio dispositivo (base de datos local Drift/SQLite), permitiendo que la app funcione sin conexión. Cuando hay red disponible, se sincronizan con un backend propio (Django REST Framework + PostgreSQL) desplegado en Railway (railway.app). Las comunicaciones entre la app y el backend se realizan siempre mediante HTTPS.

Cada usuario solo puede leer y modificar sus propias tareas y categorías; el backend filtra todas las consultas por la cuenta autenticada, sin acceso cruzado entre usuarios.

4. Finalidad del tratamiento

Los datos se usan exclusivamente para:

  • Mostrar y sincronizar tu planificador semanal entre sesiones y dispositivos.
  • Generar recordatorios locales antes de cada tarea.

Los datos no se venden, no se comparten con terceros ni se usan con fines publicitarios. La app no integra analítica ni redes publicitarias.

5. Notificaciones

Los recordatorios de tareas son 100% locales: se generan y programan en el propio dispositivo mediante las herramientas del sistema operativo Android. La app no utiliza notificaciones push remotas ni servicios de terceros para este fin.

6. Período de conservación

Los datos se conservan mientras la cuenta esté activa. El usuario puede eliminar cualquier tarea o categoría desde la propia aplicación en cualquier momento. Para eliminar completamente la cuenta y todos los datos asociados, contacta con toniketrona8@gmail.com.

7. Derechos del usuario

Puedes ejercer en cualquier momento los siguientes derechos:

  • Acceso — consultar los datos almacenados sobre ti.
  • Rectificación — corregir datos incorrectos directamente desde la app.
  • Supresión — eliminar tus datos y cuenta.

Para ejercer estos derechos, escribe a toniketrona8@gmail.com.

8. Seguridad

La autenticación se realiza mediante tokens JWT (Simple JWT), almacenados de forma cifrada en el dispositivo con flutter_secure_storage. Las contraseñas nunca se almacenan en texto plano; Django gestiona su cifrado en el servidor.

9. Cambios en esta política

Cualquier cambio relevante en esta política se comunicará mediante una actualización de la app o de esta página. La fecha de última actualización figura al inicio del documento.
