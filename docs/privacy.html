Política de privacidad

Última actualización: junio de 2025

Esta política de privacidad describe cómo la aplicación Control de Glucosa recoge, usa y protege la información personal de sus usuarios.

1. Responsable

El responsable del tratamiento de los datos es el desarrollador de la aplicación, de uso personal y educativo. Para cualquier consulta relacionada con privacidad, puedes escribir a toniketrona8@gmail.com.

2. Datos que se recogen

La aplicación recoge y almacena únicamente los datos que tú introduces:

  • Dirección de email — usada como identificador de cuenta.
  • Lecturas de glucosa — valor en mg/dL, fecha, hora y etiqueta (p.ej. "Ayunas").
  • Notas opcionales — texto libre que el usuario añade a cada lectura.

La aplicación no recoge datos de geolocalización, contactos, cámara ni ninguna información no relacionada con el seguimiento de glucosa.

3. Dónde se almacenan los datos

Los datos se almacenan en Supabase (supabase.com), una plataforma de base de datos PostgreSQL en la nube. Supabase cumple con el RGPD. Los servidores pueden estar ubicados en la Unión Europea o en Estados Unidos (dependiendo de la región del proyecto Supabase).

La protección de los datos está reforzada mediante Row Level Security (RLS): cada usuario solo puede leer y modificar sus propias lecturas; no existe acceso cruzado entre cuentas.

4. Finalidad del tratamiento

Los datos se usan exclusivamente para:

  • Mostrar el historial y estadísticas de glucosa al propio usuario.
  • Generar exportaciones en CSV o PDF a petición del usuario.
  • Enviar recordatorios locales de medición (notificaciones del dispositivo).

Los datos no se venden, no se comparten con terceros ni se usan con fines publicitarios.

5. Período de conservación

Los datos se conservan mientras la cuenta esté activa. El usuario puede eliminar cualquier lectura desde la propia aplicación en cualquier momento. Para eliminar completamente la cuenta y todos los datos asociados, contacta con toniketrona8@gmail.com.

6. Derechos del usuario

Puedes ejercer en cualquier momento los siguientes derechos:

  • Acceso — consultar los datos almacenados sobre ti.
  • Rectificación — corregir datos incorrectos directamente desde la app.
  • Supresión — eliminar tus datos y cuenta.
  • Portabilidad — exportar tus datos en CSV o PDF desde la app.

Para ejercer estos derechos, escribe a toniketrona8@gmail.com.

7. Seguridad

Las comunicaciones entre la app y Supabase se realizan siempre mediante HTTPS. Las contraseñas nunca se almacenan en texto plano; son gestionadas por el sistema de autenticación de Supabase (bcrypt). La app tiene desactivada la copia de seguridad automática de Android (android:allowBackup="false") para evitar que los datos de salud se sincronicen en copias de seguridad de terceros.

8. Cambios en esta política

Cualquier cambio relevante en esta política se comunicará mediante una actualización de la app o de esta página. La fecha de última actualización figura al inicio del documento.
